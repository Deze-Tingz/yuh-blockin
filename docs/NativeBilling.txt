 Native Billing Setup Guide

  Your App Uses RevenueCat (already integrated)

  RevenueCat wraps Apple StoreKit and Google Play Billing, handling everything for you.

  ---
  1. Google Play (Android)

  Step 1: Google Play Console

  1. Go to https://play.google.com/console
  2. Select your app → Monetize → Products

  Create Subscription:
  - Click Subscriptions → Create subscription
  - Product ID: yuh_blockin_monthly (must match exactly)
  - Price: $2.99/month
  - Billing period: 1 month

  Create One-Time Purchase:
  - Click In-app products → Create product
  - Product ID: yuh_blockin_lifetime (must match exactly)
  - Price: $19.99

  Step 2: RevenueCat Dashboard

  1. Go to https://app.revenuecat.com
  2. Create a new project (or use existing)
  3. Add App → Select Google Play
  4. Upload your Google Play Service Account JSON key
    - Get this from Google Cloud Console → Service Accounts
  5. Add your products and create entitlements:
    - Entitlement premium → attach yuh_blockin_monthly
    - Entitlement lifetime → attach yuh_blockin_lifetime
  6. Copy your Public API Key (starts with goog_)

  Step 3: Run Your App

  flutter run --dart-define=REVENUECAT_API_KEY=goog_YourKeyHere

  ---
  2. App Store (iOS)

  Step 1: App Store Connect

  1. Go to https://appstoreconnect.apple.com
  2. Select your app → Features → In-App Purchases

  Create Auto-Renewable Subscription:
  - Click + → Auto-Renewable Subscription
  - Create a Subscription Group (e.g., "Premium")
  - Reference Name: Monthly Premium
  - Product ID: yuh_blockin_monthly (must match exactly)
  - Price: $2.99

  Create Non-Consumable:
  - Click + → Non-Consumable
  - Reference Name: Lifetime Premium
  - Product ID: yuh_blockin_lifetime (must match exactly)
  - Price: $19.99

  Step 2: RevenueCat Dashboard

  1. Add App → Select App Store
  2. Enter your App-Specific Shared Secret
    - Get from App Store Connect → App → In-App Purchases → App-Specific Shared Secret
  3. Add products and map to same entitlements
  4. Copy your Public API Key (starts with appl_)

  Step 3: Run Your App

  flutter run \
    --dart-define=REVENUECAT_API_KEY=goog_YourAndroidKey \
    --dart-define=REVENUECAT_IOS_API_KEY=appl_YourIOSKey

  ---
  3. Testing

  Android Testing

  1. Google Play Console → Setup → License testing
  2. Add your Gmail addresses
  3. These accounts can make test purchases without being charged

  iOS Testing

  1. App Store Connect → Users and Access → Sandbox Testers
  2. Create sandbox test accounts
  3. Sign out of App Store on device, sign in with sandbox account

  ---
  Quick Reference

  | Item                 | Value                |
  |----------------------|----------------------|
  | Monthly Product ID   | yuh_blockin_monthly  |
  | Lifetime Product ID  | yuh_blockin_lifetime |
  | Monthly Price        | $2.99                |
  | Lifetime Price       | $19.99               |
  | Premium Entitlement  | premium              |
  | Lifetime Entitlement | lifetime             |

  ---
  Build for Production

  # Android APK
  flutter build apk --dart-define=REVENUECAT_API_KEY=goog_YourKey

  # Android App Bundle (Play Store)
  flutter build appbundle --dart-define=REVENUECAT_API_KEY=goog_YourKey

  # iOS
  flutter build ios --dart-define=REVENUECAT_IOS_API_KEY=appl_YourKey

  That's it! RevenueCat handles receipt validation, subscription status, and cross-platform syncing automatically.
